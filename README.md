# Tesla & Amazon Stock Analysis

## Overview

This project analyzes Tesla and Amazon stock prices and annual revenues.

The project includes:

- Stock price extraction using yfinance
- Revenue web scraping using BeautifulSoup
- Data cleaning with Pandas
- Interactive dashboards using Plotly
- Revenue growth analysis

## Libraries

- pandas
- plotly
- yfinance
- requests
- beautifulsoup4
