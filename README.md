# Tesla & Amazon Financial Analysis

## Project Overview

This project analyzes the historical stock prices and revenues of Tesla and Amazon using Python.

The objective is to compare the financial performance of both companies through data collection, cleaning, analysis, and visualization.

---

## Dataset

The project includes:

- Tesla historical stock prices
- Amazon historical stock prices
- Tesla quarterly revenue
- Amazon quarterly revenue

---

## Tools & Libraries

- Python
- Pandas
- BeautifulSoup
- yfinance
- Matplotlib

## Project Workflow

1. Download stock price data
2. Scrape revenue data
3. Clean datasets
4. Visualize stock trends
5. Compare financial performance

The project demonstrates the relationship between stock price trends and revenue growth for Tesla and Amazon using interactive visualizations.
